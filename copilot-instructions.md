## 🧩 ikas Component Architecture

### Component Pattern (IMPORTANT!)

**ikas bileşenlerinde CSS Modules veya styled-jsx ÇALIŞMAZ!**  
Tüm stiller **inline styles** (CSSProperties) ile yazılmalıdır.

```tsx
// ✅ DOĞRU - Inline Styles
import { observer } from "mobx-react-lite";
import { useStore } from "@ikas/storefront";
import { CSSProperties } from "react";

const styles: { [key: string]: CSSProperties } = {
  container: {
    display: "flex",
    padding: "24px",
    background: "white",
    borderRadius: "16px",
  },
  title: {
    fontSize: "20px",
    fontWeight: 600,
    color: "#111827",
  },
};

const MyComponent: React.FC<Props> = (props) => {
  const store = useStore();
  
  return (
    <div style={styles.container}>
      <h2 style={styles.title}>Title</h2>
    </div>
  );
};

export default observer(MyComponent);
```

```tsx
// ❌ YANLIŞ - styled-jsx (Çalışmaz!)
const MyComponent = () => {
  return (
    <div className="container">
      <style jsx>{`
        .container { padding: 24px; }  // Bu çalışmaz!
      `}</style>
    </div>
  );
};
```

### observer() Wrapper (ZORUNLU)
MobX reaktivitesi için tüm bileşenler `observer()` ile sarılmalıdır:

```tsx
import { observer } from "mobx-react-lite";

// Named export
const MyComponent = observer(() => { ... });

// Default export
export default observer(MyComponent);
```

## 📚 ikas Store API Reference

### Importing ikas
```tsx
import { useStore, IkasOrderLineItem, IkasImage, IkasCart } from "@ikas/storefront";
```

### CartStore API

#### Sepet Verisi Okuma
```tsx
const store = useStore();
const cart = store.cartStore.cart;

// Sepet bilgileri
cart?.orderLineItems        // Sepetteki ürünler (IkasOrderLineItem[])
cart?.itemQuantity          // Toplam ürün sayısı
cart?.totalFinalPrice       // Toplam tutar (number)
cart?.formattedTotalFinalPrice  // Formatlanmış tutar ("£29.99")
cart?.couponCode            // Uygulanan kupon kodu
```

#### Sepet İşlemleri
```tsx
// Ürün silme
await store.cartStore.removeItem(item);

// Miktar güncelleme
await store.cartStore.changeItemQuantity(item, newQuantity);

// Kupon uygulama
const result = await store.cartStore.saveCouponCode(code);
if (!result.success) {
  // Hata: Geçersiz kupon
}

// Kupon kaldırma
await store.cartStore.removeCouponCode();
```

#### IkasOrderLineItem Properties
```tsx
item.id                                 // Benzersiz ID
item.quantity                           // Miktar
item.variant?.mainImage?.src            // Ürün resmi URL
item.variant?.name                      // Ürün adı
item.variant?.sku                       // SKU kodu
item.formattedFinalPriceWithQuantity    // Formatlanmış fiyat
item.formattedPriceWithQuantity         // Alternatif fiyat
item.options                            // Varyant seçenekleri
  ├── .name                             // Seçenek adı ("Color")
  └── .values                           // Değerler array'i
```

### Checkout

**ÖNEMLİ:** Checkout sayfası için ikas'ın default checkout'unu kullanıyoruz:

```tsx
// pages/checkout.tsx
import { CheckoutPage } from "@ikas/storefront";
import { CheckoutPage as CheckoutPageNext } from "@ikas/storefront-next";

const PageComponent = CheckoutPage.default;

const Page = (props: any) => {
  return <PageComponent {...props} />;
};

export default Page;
export const getStaticProps = CheckoutPageNext.getStaticProps;
```

```tsx

<Link href="/checkout">
```

**Yapı:** ikas CartPage + Custom Wrapper

```tsx
// pages/cart.tsx
import { CartPage } from "@ikas/storefront";
import { CartPage as CartPageNext } from "@ikas/storefront-next";
import Components from "src/components/__generated__/pages/cart";

const PageComponent = CartPage.default;

const Page = (props: any) => {
  return (
    <>
      <style jsx global>{`
        .cart-page-wrapper {
          min-height: 100vh;
          background: linear-gradient(165deg, #0d9488 0%, #0891b2 50%, #06b6d4 100%);
          padding-top: 80px;
        }
        .cart-page-container {
          max-width: 1200px;
          margin: 0 auto;
          padding: 24px;
          display: grid;
          grid-template-columns: 1fr;
          gap: 24px;
        }
        @media (min-width: 1024px) {
          .cart-page-container {
            grid-template-columns: 1fr 400px;
            gap: 40px;
            padding: 40px;
          }
        }
      `}</style>
      <div className="cart-page-wrapper">
        <div className="cart-page-container">
          <PageComponent components={Components} {...props} />
        </div>
      </div>
    </>
  );
};

export default Page;
export const getStaticProps = CartPageNext.getStaticProps;
```

### Code Style
- Use TypeScript for all new files
- Prefer functional components with hooks
- Use "use client" directive for client components

- Use Tailwind utility classes (avoid custom CSS)

### File Organization
- Components in `/components` or `/components/ui`

- Utils in `/lib`

### Performance Best Practices
- Use `useRef` for animation state (avoid re-renders)
- Implement RAF for continuous animations
- Lazy load images with next/image (when implemented)
- Memoize expensive calculations
- Use passive event listeners where possible

### Accessibility
- Semantic HTML elements
- Keyboard navigation support
- ARIA labels on interactive elements
- Focus states on all interactive elements
